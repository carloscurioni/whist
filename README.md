# Whist

Este projeto contém uma página HTML estática, pronta para ser aberta localmente no navegador ou publicada em uma URL pública.

## Arquivo principal

- `whist_1.html` — página principal do projeto

## Como visualizar localmente

Há duas formas simples de abrir o arquivo:

### Opção 1: abrir diretamente no navegador
Basta dar duplo clique no arquivo `whist_1.html`.

### Opção 2: usar um servidor local
Isso é recomendado caso a página utilize scripts, fontes, imagens ou recursos relativos.

Exemplo com Python:

```bash
python3 -m http.server 8000
